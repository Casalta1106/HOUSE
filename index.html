Configure o Supabase para salvar e restaurar seus dados.
📋 Como configurar
1. Conta gratuita em supabase.com
2. Project Settings → API → copie URL e anon key
3. Execute no SQL Editor:

CREATE TABLE IF NOT EXISTS house_backups (id uuid DEFAULT gen_random_uuid() PRIMARY KEY, user_id uuid REFERENCES auth.users NOT NULL, label text NOT NULL, payload jsonb NOT NULL, created_at timestamptz DEFAULT now());
ALTER TABLE house_backups ENABLE ROW LEVEL SECURITY;
CREATE POLICY "own" ON house_backups USING (auth.uid() = user_id) WITH CHECK (auth.uid() = user_id);
